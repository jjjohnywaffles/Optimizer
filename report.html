What "Consider Caching Repeated Computation" Means

Repeated computations inside loops can significantly impact runtime performance. Caching stores the result of a computation so it doesnâ€™t need to be recalculated multiple times. For example: 

# Instead of:
for i in range(100):
    result = expensive_function(i)

# Use:
cache = {}
for i in range(100):
    if i not in cache:
        cache[i] = expensive_function(i)
    result = cache[i]
